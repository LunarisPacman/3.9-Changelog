# Lunaris v3.9 | Android 16 QPR2

**Release Date:** April 2, 2026  
**Supported Devices:** Nothing Phone (2a) [Pacman] | Nothing Phone (2a) Plus [PacmanPro]

---

## 🆕 What's New

- Improved haptics for a crisper feel
- Improved app opening/closing animations and overall speed
- Marginally improved gaming performance
- Included ViperFx for better audio quality
- Fixed charging threshold
- iOS emojis by default
- Fixed face unlock reliability issue
- Fixed Quick Settings lag when having too many notifications
- [Source changes](https://t.me/lunaris_update/209)

---

## ⚠️ Important Notes

- GApps are included by default
- Fenrir is not included by default
- Play Integrity: **Strong** with Fenrir (as of April 2, 2026)
- **Clean flash required for 2a Plus (PacmanPro)**
- Dirty flash/local update is supported if you are currently on Lunaris v3.8
